# Stock-Market-App
Applied: 
- Clean architectural
- Dependency injection with Dagger-Hilt
- SOLID principles
- CSV parsing with OpenCSV
- Working with remote APIs using Retrofit
- Local caching with Room
- Custom drawing on a canvas using Compose
- Version Catalog with toml file
